# my-voice — 팀 형식 우선, 내 말투 덧입히기

입력(텍스트 또는 쓰려는 의도 + 대상 매체)을 받아 **초안**을 만든다. 전송·게시는 하지 않는다.

## 우선순위 (절대 규칙)

**형식·구조 = 팀 컨벤션이 항상 우선. 말투 = 표현층에만.**
팀 규칙이 정한 구조(PR 제목/커밋 접두어/섹션 순서/필수 항목/언어)는 개인 말투로 바꾸지 않는다.
개인 말투는 "그 골격 안에서 문장을 어떻게 쓰는가"에만 적용한다. 충돌하면 팀 규칙이 이긴다.

## 1. 대상 매체 확인

무엇을 쓰는지 먼저 정한다: PR 설명 / PR·리뷰 코멘트 / Slack 메시지 / 기타. 불명확하면 사용자에게 묻는다.

## 2. 팀 형식 먼저 참조 (런타임 탐색 — 하드코딩 금지)

대상에 맞는 팀 스킬·컨벤션을 **먼저 찾아 형식을 그대로 따른다**. 매체별로:

- **PR 설명**: 팀 `create-pr-skill`(있으면)의 PR 본문 형식·커밋 접두어·base 규칙. 없으면 작업 중인 repo의 `CLAUDE.md`·`.claude/rules/github-workflow.md` 참조.
- **PR·리뷰 코멘트**: 팀 `pr-comment-skill`의 코멘트 형식, repo `CLAUDE.md`의 리뷰 규칙(예: laftel — 리뷰 코멘트 한국어, 인라인 답글, Codex `@codex` 멘션 등).
- **Slack**: 채널 관례가 있으면(예: `[OTT PR]` 키워드, 스레드 규칙) 따른다.

탐색 방법: 현재 세션의 사용 가능 스킬 목록·`CLAUDE.md`·`.claude/rules/`를 확인한다. 팀 형식 규칙이 있으면 **그것이 구조의 정본**이다. 없으면 상식적 형식.

## 3. 내 말투 프로필 로드

`~/ryu-mg/wiki/concepts/bao-slack-writing-style.md`를 읽는다. 여기 규칙(종결어미·문장부호 습관·"요-" 지시어·감탄사·완곡 요청·이중 레지스터)을 **표현층에만** 적용한다.

## 3-1. 라프텔 말투 컨텍스트 (평어 기본)

작업 맥락이 라프텔이면 아래를 §3 위에 얹는다.

- 라프텔 사내 **기본 말투는 평어**다 (직급 무관 평어체 — `~해`, `~하자`, `~했어`, `~할게`). 동료 간 Slack·구두 소통 기본값.
- bao 본인은 **평어와 존대(해요체)를 상황 따라 적당히 섞어** 쓴다. 평어로만 딱딱하게도, 존대로만 과공손하게도 안 쓴다.
  - 팀 내부 캐주얼(스레드 잡담, 동료 대상) → 평어 쪽.
  - 타 팀·상위 공지·공식 알림·외부 → 존대(해요체/합니다체) 쪽.
- 프로필(§3)의 해요체 예시는 **존대 레지스터일 때** 표현이다. 평어로 쓸 땐 **종결어미만 평어로 내리고**(`~해요`→`~해`, `~할게요`→`~할게`, `~같아요`→`~같아`), 나머지 습관(요-지시어·`,,`·`~`·완곡 요청·이모지)은 그대로 유지한다.
- 레지스터가 불명확하면 사용자에게 평어/존대 중 무엇으로 쓸지 묻는다.

## 4. 초안 작성

- 팀 형식 골격(2) 안에 내용을 배치하고, 문장 표현에 말투(3)를 입힌다.
- **이중 레지스터 존중**: 기술 내용은 구조적·정확하게(bullet/asis-tobe/굵은 요약), 포장만 친근하게.
- 매체별 톤 조정:
  - Slack 캐주얼 채널 → 말투 풀로(요-지시어·`~`·`,,`·늘린 어미·이모지).
  - PR 설명·공식 알림 → 말투는 옅게. 팀이 정한 `~합니다`체·구조 우선, 개인색은 과하지 않게.
  - 리뷰 코멘트 → 팀 규칙(한국어 등) 지키며 완곡·저자세 요청 톤만 살짝.
- 팀 규칙과 말투가 부딪히면(예: 팀은 `~합니다`체 강제) 팀 규칙을 따르고, 그 사실을 한 줄로 알린다.

## 5. 제시 (전송 안 함)

- 초안을 보여주고 끝낸다. 게시·전송·PR 생성은 하지 않는다.
- 실제 전송이 필요하면 해당 팀 스킬(create-pr-skill, slack 전송 등)로 넘기라고 안내. 이 스킬은 **초안 생성 전용**.

## 금기

- 팀이 정한 구조·접두어·언어·필수 항목을 개인 말투로 훼손 금지.
- 사용자 확인 없이 전송/게시/PR 생성 금지.
- 말투 프로필을 임의로 재정의 금지(갱신은 `update-my-style` 담당).
